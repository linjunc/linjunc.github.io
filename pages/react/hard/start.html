前言 #

Hello，大家好，我是小丞同学！半年时间，带来了一个新的专栏：React 源码解析
这段时间，一直在深入 React 源码学习，学习了卡颂老师的 React 技术揭秘、外星人老师的 React 进阶实践指南小册等优秀的博文。最终结合自己的理解和思考，写下了这个专栏，希望我的理解能够为你的源码学习之路带来一定的启发。
文中内容如有错误之处，欢迎指出，我们共同成长，一起进步。
